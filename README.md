Tech used

This is a single static HTML file — no build tools, no frameworks, no installation needed. Everything (HTML, CSS, JavaScript) is in one file: index.html.

This makes it extremely easy to host anywhere, including free platforms like Netlify, Vercel, or GitHub Pages.

How to view it locally


Download or clone this repository
Open index.html in any web browser (just double-click it)


No server or installation required.

How to deploy / update the live website

This site is hosted on Netlify.

To publish updates after making changes:


Edit index.html with your changes
Go to the Netlify project dashboard → Deploys tab
Drag and drop the updated index.html file into the deploy box
The live site updates within seconds — the website link stays the same


To connect this GitHub repo to Netlify for automatic deploys (optional, recommended later):


In Netlify, choose "Import from GitHub" instead of manual deploy
Select this repository
Every time this repo is updated on GitHub, Netlify will automatically redeploy the site
